## 🎯 项目介绍

**VoiceDraw AI** 是一款专为语音交互场景设计的AI绘图工具。用户全程无需使用鼠标或键盘，仅通过**中文语音指令**即可完成从基础图形绘制到颜色填充、样式调整、撤销重置等完整绘图创作流程。

本项目的核心目标是探索**语音作为创作工具的可行性**，在指令理解准确性、响应延迟、复杂指令拆解三个维度上进行综合优化。同时预留了**AI文生图接口**，支持后续接入大模型实现语音描述生成图像的能力。

### 创作场景

- 🗣️ 说出"画圆" → 切换圆形模式 → 在画布上用手指/触控笔绘制
- 🎨 说出"红色" → 画笔颜色切换为红色
- 📏 说出"加粗" → 线条宽度增加
- ↩️ 说出"撤销" → 回退到上一步操作

### 项目亮点

| 维度 | 实现方式 |
|------|---------|
| **零依赖** | 纯浏览器原生API，无任何第三方框架/CDN |
| **即开即用** | 单HTML文件，双击即可运行 |
| **中文优化** | 针对中文语音特点进行关键词提取与模糊匹配 |
| **容错设计** | 支持多种近义表达、模糊匹配、多候选结果 |
| **即时反馈** | 语音合成(TTS)即时确认操作结果 |

## 🚀 快速开始

### 运行方式

**方式一：直接打开（推荐）**
1. 下载 `index.html` 文件
2. 使用 **Google Chrome** 浏览器双击打开
3. 点击麦克风按钮 🎙️
4. 允许浏览器使用麦克风权限
5. 开始说出绘图指令！

**方式二：本地服务器**
```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .

# 然后访问 http://localhost:8080
```

### 环境要求

- **浏览器**: Google Chrome 80+（推荐）、Microsoft Edge 80+
- **权限**: 麦克风访问权限
- **网络**: 首次使用时需要网络连接（Chrome语音识别服务）

> ⚠️ **重要**: Web Speech API 的语音识别功能在 Chrome 浏览器中需要网络连接，因为它依赖 Google 的云端语音识别服务。Firefox 和 Safari 目前对中文语音识别的支持有限。

## 📋 使用说明

### 基本流程

1. **启动识别**: 点击麦克风按钮或说"开始"
2. **选择模式**: 说"画圆"/"画矩形"/"自由画"等
3. **选择颜色**: 说"红色"/"蓝色"等
4. **开始绘制**: 在画布上用鼠标/触控笔绘制
5. **调整样式**: 说"加粗"/"变细"
6. **撤销修改**: 说"撤销"
7. **导出作品**: 说"导出"

### 指令示例

```
"画圆"                    → 切换圆形绘制模式
"红色"                    → 切换画笔为红色
"画一个蓝色的矩形"          → 切换矩形模式 + 蓝色
"加粗"                    → 线宽+2px
"填充"                    → 点击区域填色
"撤销"                    → 回退一步
"清空"                    → 清除画布
"导出"                    → 保存PNG
"帮助"                    → 语音播报所有指令
```

### 核心技术选型

| 模块 | 技术 | 选择理由 |
|------|------|---------|
| 语音识别 | `SpeechRecognition` (Web Speech API) | 浏览器原生支持，无需第三方库，中文识别准确率较高 |
| 语音合成 | `SpeechSynthesis` (Web Speech API) | 与识别同源API，提供中文TTS反馈 |
| 绘图渲染 | `Canvas 2D API` | 浏览器原生，性能优异，支持像素级操作 |
| 撤销机制 | `ImageData` 快照栈 | 简单可靠，每步保存完整画布状态 |
| 填充算法 | 洪水填充 (Flood Fill) | 经典4连通域算法，基于像素颜色容差匹配 |
| 指令解析 | 关键词匹配 + 字符重叠度模糊匹配 | 轻量级NLP方案，无需加载模型，响应即时 |

## 📦 依赖列表

### 运行时依赖

**本项目为零外部依赖（Zero-Dependency）纯浏览器原生应用。**

所有功能均基于以下浏览器原生API实现：

| API | 用途 | W3C标准 |
|-----|------|---------|
| `SpeechRecognition` | 语音识别（中文） | [Web Speech API](https://wicg.github.io/speech-api/) |
| `SpeechSynthesis` | 语音合成反馈 | [Web Speech API](https://wicg.github.io/speech-api/) |
| `Canvas 2D Context` | 绘图渲染 | [HTML Canvas 2D Context](https://html.spec.whatwg.org/multipage/canvas.html) |
| `ImageData` | 像素级画布操作（撤销/填充） | [HTML Living Standard](https://html.spec.whatwg.org/) |
| `fetch` | HTTP请求（AI预留接口） | [Fetch API](https://fetch.spec.whatwg.org/) |

### 开发/构建依赖

**无。** 本项目为纯静态HTML文件，无需编译、打包、转译。

### 原创功能说明

以下功能为本项目**完全原创实现**，未引用或复用任何第三方代码：

1. **指令解析引擎**: 双层匹配（关键词+模糊）的中文语音指令解析器
2. **绘图模式管理器**: 自由绘制/直线/圆/矩形/填充/擦除六种模式的统一管理
3. **洪水填充算法**: 基于像素颜色容差的4连通域填充实现
4. **撤销快照系统**: 基于ImageData的增量撤销栈
5. **语音-视觉双反馈系统**: TTS语音反馈 + 画布文字叠加 + 指令日志
6. **复合指令拆解器**: 单句多意图识别（如"画一个红色的圆"）
