#  VoiceDraw AI — 语音控制数据可视化工具

>  Demo视频: https://b23.tv/qHsCWYM

> 通过中文语音指令创建并编辑柱状图、饼图、折线图、流程图，支持4种风格一键切换。

##  快速开始
1. 用 Chrome 打开 index.html，点击麦克风授权
2. 说 画柱状图 跟随语音引导完成创建
3. 图表完成后说 导出 保存为PNG

##  核心功能
- 柱状图/饼图/折线图/流程图 四种图表语音创建
- 暖色风/商务风/科技风/简约风 四种风格一键切换
- 标题/标签/柱子的颜色、大小、粗细独立编辑
- 分步引导式图表创建向导
- 3x高清导出（保持当前风格）
- 像素快照撤销栈
- 柱子宽度独立锁定

##  主要指令
创建: 画柱状图 画饼图 画折线图 画流程图
修改: 将标题改为 将标题放大/缩小/加粗 将第N个标签改为 将所有标签改为蓝色
柱子: 将第N根柱子改为金色 将柱子加粗/变细
流程图: 将第N个节点名称改为 放大节点框 箭头变长/加粗
风格: 暖色风 商务风 科技风 简约风
操作: 撤销 清空 重置 导出 帮助

##  技术栈
Web Speech API + Canvas 2D + 零外部依赖 + Chrome 80+
